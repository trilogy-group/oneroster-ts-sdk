# DemographicsManagement
(*demographicsManagement*)

## Overview

This enables the management of demographics information (each assigned to a specific user). The sourcedIds for the user and the demographics records should be the same.

### Available Operations

* [getAllDemographics](#getalldemographics) - Get all Demographic records
* [postDemographics](#postdemographics) - Create a new Demographic record
* [getDemographics](#getdemographics) - Get a specific Demographic record
* [putDemographics](#putdemographics) - Update a Demographic record
* [deleteDemographics](#deletedemographics) - Delete a Demographic record

## getAllDemographics

To get all Demographic records on the service provider.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.demographicsManagement.getAllDemographics({
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    console.log(page);
  }
}

run();
```
