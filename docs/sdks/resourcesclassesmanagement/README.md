# ResourcesClassesManagement
(*resourcesClassesManagement*)

## Overview

This enables the management of resources for classes.

### Available Operations

* [getResourcesForClass](#getresourcesforclass) - Get resources for a class
* [assignResourceToClass](#assignresourcetoclass) - Assign a resource to a class
* [removeResourceFromClass](#removeresourcefromclass) - Remove a resource from a class

## getResourcesForClass

To get the collection of resources available to a specific class. If the corresponding record cannot be located, the api will return a 404 error code and message 'Class not found.'

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.resourcesClassesManagement.getResourcesForClass({
    classSourcedId: "<id>",
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    console.log(page);
  }
}

run();
```

## assignResourceToClass

To assign an existing resource to the specified class. The responding system must return the set of sourcedIds that have been allocated to the newly created resource records.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.resourcesClassesManagement.assignResourceToClass({
    classSourcedId: "<id>",
    requestBody: {
      resource: {
        sourcedId: "<id>",
      },
    },
  });

  console.log(result);
}

run();
```
