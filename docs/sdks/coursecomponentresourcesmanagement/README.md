# CourseComponentResourcesManagement
(*courseComponentResourcesManagement*)

## Overview

This enables the management of Course Component Resources i.e. resources for a course component.

### Available Operations

* [createComponentResource](#createcomponentresource) - Create Component Resource
* [getAllComponentResources](#getallcomponentresources) - Get all Component Resources
* [getComponentResource](#getcomponentresource) - Get a specific Component Resource
* [putComponentResource](#putcomponentresource) - Update a Component Resource
* [deleteComponentResource](#deletecomponentresource) - Delete a Component Resource

## createComponentResource

To create a new Component Resource. The responding system must return the set of sourcedIds that have been allocated to the newly created componentResource record.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.courseComponentResourcesManagement.createComponentResource();

  console.log(result);
}

run();
```

    fields: "sourcedId,name",
    filter: "status='active'",
