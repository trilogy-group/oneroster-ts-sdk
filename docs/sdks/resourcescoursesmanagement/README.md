# ResourcesCoursesManagement
(*resourcesCoursesManagement*)

## Overview

This enables the management of resources for courses.

### Available Operations

* [getResourcesForCourse](#getresourcesforcourse) - Get resources for a course
* [assignResourceToCourse](#assignresourcetocourse) - Assign a resource to a course
* [removeResourceFromCourse](#removeresourcefromcourse) - Remove a resource from a course

## getResourcesForCourse

To get the collection of resources assigned to a specific course. If the corresponding record cannot be located, the api will return a 404 error code and message 'Course not found.'

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.resourcesCoursesManagement.getResourcesForCourse({
    courseSourcedId: "<id>",
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    console.log(page);
  }
}

run();
```

## assignResourceToCourse

### Example Usage

```typescript

const oneRoster = new OneRoster({
  oAuth2: process.env["ONEROSTER_O_AUTH2"] ?? "",
});

async function run() {
  const result = await oneRoster.resourcesCoursesManagement.assignResourceToCourse({
    courseSourcedId: "<id>",
    requestBody: {
      resource: {
        sourcedId: "<id>",
      },
    },
  });

  console.log(result);
}

run();
```
