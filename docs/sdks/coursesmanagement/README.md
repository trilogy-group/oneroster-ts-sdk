# CoursesManagement
(*coursesManagement*)

## Overview

This enables the management of Courses i.e. programme of study.

### Available Operations

* [getAllCourses](#getallcourses) - Get All Courses
* [createCourse](#createcourse) - Create a Course
* [getClassesForCourse](#getclassesforcourse) - Get Classes for a Course
* [getCourse](#getcourse) - Get a specific Course
* [putCourse](#putcourse) - Update a Course
* [deleteCourse](#deletecourse) - Delete a Course
* [createComponentResource](#createcomponentresource) - Create Component Resource
* [getAllComponentResources](#getallcomponentresources) - Get all Component Resources
* [getComponentResource](#getcomponentresource) - Get a specific Component Resource
* [putComponentResource](#putcomponentresource) - Update a Component Resource
* [deleteComponentResource](#deletecomponentresource) - Delete a Component Resource
* [createCourseComponent](#createcoursecomponent) - Create Course Component
* [getAllCourseComponents](#getallcoursecomponents) - Get all Course Components
* [getCourseComponent](#getcoursecomponent) - Get a specific Course Component
* [putCourseComponent](#putcoursecomponent) - Update a Course Component
* [deleteCourseComponent](#deletecoursecomponent) - Delete a Course Component
* [getCoursesForSchool](#getcoursesforschool) - Get all Courses for a School

## getAllCourses

To get all Courses on the service provider.

### Example Usage

```typescript

  security: {
    clientID: process.env["ONEROSTER_CLIENT_ID"] ?? "",
    clientSecret: process.env["ONEROSTER_CLIENT_SECRET"] ?? "",
  },
});

async function run() {
  const result = await oneRoster.coursesManagement.getAllCourses({
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    // Handle the page

}

run();
```
